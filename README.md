# Fantom Staré Prahy

Jednoduchá webová aplikace pro zjednodušení otravného počítání tokenů ve hře Fantom Staré Prahy

Pro spuštěné otevřete následující odkaz:

https://htmlpreview.github.io/?https://github.com/Jakuje/fantom-stare-prahy/blob/main/index.html

Nebo si stáhněte HTML soubor na svůj počítač a otevřete jej v prohližeči.


Aplikace je optimalizovaná pro mobilní telefon. Pro použití na PC je dobré stráhku trochu přiblížit.
